# Dr. Geison Frank — Landing Page

Landing page de alta conversão para o Dr. Geison Frank (Odontopediatria e Pacientes Especiais).
Site estático de arquivo único (`index.html`) — sem build necessário.

## Antes de publicar
- Confirmar o número de WhatsApp na constante `WHATSAPP` no topo do `index.html`.
- Preencher `CRO/PB [XXXX]` no rodapé.
- Ajustar os números dos selos de confiança (ex.: "+10 anos") para os valores reais.

## Deploy
Site estático. Na Vercel, basta importar o repositório — sem framework, output = raiz.
